# Oboy

This a Gameboy emulator written in the Odin programming langaue (with Raylib).

## Features
- [x] Joypad
- [x] Background
- [x] Objects
- [x] Instruction set
- [ ] Window
- [ ] Audio
- [ ] Swap Rom
- [ ] Input Remaps
- [ ] Color Palette
- [ ] Bring your own Boot Rom

## Joypad

| Button/Aktion | Key |
| ----------- | ----------- |
| Left | A|
| Right | D|
| Up | W|
| Down | S|
| B | Q|
| A | E|
| Select | Enter|
| Start | Space|
